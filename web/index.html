Expose.dart

HTML Presentations Made Easy

Created by Guillaume Girou / @GirouGuillaume

Heads Up

expose.dart is a framework for easily creating beautiful presentations using HTML. You'll need a browser with support for CSS 3D transforms to see it in its full glory.

Oh hey, these are some notes. They'll be hidden in your presentation, but you can see them if you open the speaker notes window (hit 's' on your keyboard).
